Inicialização do caso de uso:
        Para rodar e testar o  caso de uso, primeiro é necessário  ter completado o caso de uso 1,
    ou seja, é necessário que  uma comanda já tenha sido criada e  inserida no sistema. Para  tal,
    pode-se apenas  seguir os passos de criação  de comanda instruidos,  sendo que ao  final basta
    apertar no botão "Retornar à janela principal".
        Tendo feito isso, uma lista de comandas ativas irá automaticamente ser mostrada na página,
    logo, para editar uma comanda em específico, basta apertar o botão "Editar Comanda" ao lado da
    que se deseja.
        Finalmente,  se o programa  for redirecionado  à outra  página significa que a execução do
    caso foi um sucesso, caso contrário houve algum tipo de problema em se conectar com o banco de
    dados onde são guardadas as comandas ativa.
