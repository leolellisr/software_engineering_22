Inicialização do caso de uso:
        Para rodar e testar o caso de uso, basta rodar o arquivo executável start.sh, que por
    sua vez irá  chamar  os  scripts  mongo.sh e  node.sh.  Feito isso,  basta entrar na  url 
    localhost://3000. Na página web aberta, digite um número à escolha no campo de criação de
    comanda, e em seguida aperte no botão "Criar Comanda".
        Caso ocorra o redirecionamento para uma outra página web, o caso de uso de criação de
    comanda terá sido um sucesso, caso contrário,  terá ocorrido  algum erro no  processo  de
    adicionar uma nova comanda ao banco de dados.
