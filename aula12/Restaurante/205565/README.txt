Para rodar a aplicação, utilize o arquivo start.sh.
Entre em localhost:3000. À esquerda você verá uma lista das comandas existentes, tente criar uma nova comanda.
Você irá listar as novas comandas sendo adicionadas. Clique em Revisão para rever os itens da comanda e terminar o pedido. Você pode escolher duas formas de pagamento, via cartão e dinheiro. Caso escolha dinheiro, é necessário incluir um saldo suficientemente grande para o pagamento.
Ao finalizar, você será levado a uma página de conclusão. Clique no botão para voltar para a página inicial. Note que a comanda finalizada foi excluida.
